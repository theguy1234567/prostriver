# ProStriver

ProStriver is a productivity-focused web platform designed to help users build consistent learning habits through structured revision, task tracking, and performance analytics.

This is a collaborative team project built to create a practical and efficient learning workflow for users.

---


## Features

* Authentication with OTP-based email verification
* Daily learning and revision tracking
* Analytics dashboard for performance insights
* Task completion and progress tracking
* Revision scheduling system
* Fully responsive UI for mobile and desktop

---

## Purpose

ProStriver aims to help users stay consistent with their learning, manage revision effectively, and build discipline through structured tracking.
